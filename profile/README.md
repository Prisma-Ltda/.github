# Prisma - Feedback Application 

## 📌 Descrição
O Prisma é uma plataforma web voltada à coleta, consolidação e análise de feedbacks internos entre gestores, utilizando o modelo Net Promoter Score (NPS) aliado à análise de sentimento por Inteligência Artificial.

A solução permite centralizar a gestão do clima organizacional, transformando dados em insights estratégicos para apoiar a tomada de decisão e o desenvolvimento dos colaboradores.

---

## 🎯 Objetivo
Desenvolver uma plataforma capaz de coletar e analisar feedbacks internos, gerando indicadores e insights automáticos por meio de Inteligência Artificial.
